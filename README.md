There are two tiny DirectX Renderer for learning Graphics API --- DX12 Soft Renderer and DXR Renderer

The DX12 Soft Renderer is following with <<Introduction to 3D Game Programming with DirectX12>> from Frank D.Luna.
From now, I have learned a half of this book(chapter 12), accordingly, the functions of this renderer are also only updated in these chapter.
Such as: Fundamental illumination, shadow map, stencil test and so on.

And the DXR Renderer is following with the tutorial of Nvida:
https://developer.nvidia.com/rtx/raytracing/dxr/dx12-raytracing-tutorial-part-1
This renderer is rougher than the the last one, it only achieve illumination and shadow cast.
